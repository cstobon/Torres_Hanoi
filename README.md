Torres de Hanoi. 

Resolución del juego de torres de Hanoi en lenguaje de C++.

La solución se hace a traves de una estructura de datos Pila generada a través de clases en C++.
Todas las clases se separan en sus declaraciones (.h) y sus definiciones (.cpp).
La rutina para la resolucion del problema de torres de hanoi esta en Hanoi.cpp.

Compilacion:  g++ -std=c++11 hanoi.cpp pila.cpp nodo.cpp dato.cpp
Archivo ejecutable: a.out
